Integrate your Woocommerce store with Kinguin marketplace. This tool allows you to import games from Kinguin into your Woocommerce shop.

== Description ==

Import over 70,000 digital products to your online store, including video games, software, gift cards and in-game content.

== Features ==

* 70,000 digital products, including:
** Video game keys for numerous platforms, including Steam, Origin, Uplay or GOG.
** Gift cards for popular CS:GO stores, including Gamdom, DuelBits or CSGO Empire
** Computer software, including Windows, antivirus or security software
* Automated key-delivery via email
* Support 24/7
* Free of charge
* Full automation
* Special discounts on digital products
* Ready products’ descriptions and images
* Automated or manual pricing options


== Installation Instructions ==

1. Upload `kinguin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

How to connect Woocommerce with Kinguin marketplace:

1. Set up your API Key from Kinguin personal store dashboard and save webhooks in store's settings
2. Import products from Kinguin into your Woocommerce shop.
